## 🎙️ About Sambhasha Media Day
**Sambhasha** is the flagship inter-school media competition and media day organized by the **Nalanda College Communication Unit (NCCU)**. Spanning over two decades of excellence, it serves as the ultimate platform for young media enthusiasts across the island to showcase their talents in broadcasting, journalism, and creative media production.

## 💻 The System
To streamline the registration and coordination of hundreds of schools and thousands of contestants, we have engineered a custom **Full-Stack Management Portal**. This system acts as the central nervous system for Sambhasha XXVI, ensuring precision, security, and real-time data handling.

### Key Technical Features
* **Secure School Authentication**: A robust login/signup system integrated with Supabase Auth and Resend SMTP for professional email verification.
* **Contestant Management**: Dynamic entry forms allowing schools to manage their participants across various categories with ease.
* **Advanced Data Architecture**: A Relational Database Management System (RDBMS) using PostgreSQL to ensure data integrity.
* **Real-time Logging**: A custom-built system log monitoring actions and ensuring auditability across the platform.
* **Elite UI/UX**: A dark-themed, minimalist interface built with React and Vite, optimized for high performance and clarity.

### Tech Stack
* **Frontend**: React.js, Vite,Typescript, Tailwind CSS
* **Backend/Database**: Supabase (PostgreSQL)
* **Hosting**: Vercel
* **Infrastructure**: GitHub Actions, Resend SMTP
